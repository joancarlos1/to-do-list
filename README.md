Exercício em formato de to-do-list em ConsoleApp afim de colocar em prática tecnologias da linguagem C#.

Tecnologias usadas:

- C#

- .NET Console

- System.Text.Json = Utilizado para salvar e carregar as tarefas em tasks.json.

- System.IO

- LINQ = Utilizado para operações mais organizadas com listas, como: ordenar, filtrar, buscar tares por ID parcial.

Nesse exercício foram usados estruturas condicionais: if, foreach, switch.
Tratamento de erros por blocos usando Try e Catch.
